# Sylphx Flow

> Detailed project knowledge → `MEMORY.md`
> Daily logs → `memory/YYYY-MM-DD.md`

## Quick Commands

```bash
# Development
bun run dev:flow              # Watch mode
bun run build:flow           # Build flow package
bun test                     # Run tests

# Package management
bun run clean:all            # Clean all build artifacts
bun install                  # Install dependencies

# Linting/Formatting
bun run lint                 # Check with Biome
bun run format               # Format with Biome

# Release
bun run release              # Build and publish
```
