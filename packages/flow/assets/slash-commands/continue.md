# Continue

**Goal: Production-grade, not MVP.** No workarounds. No shortcuts. The final implementation.

## Think

不要 checklist。**深入思考**。

這個產品還缺什麼？從每一個可能的角度去想：
- 不同類型的用戶會怎樣用？他們會卡在哪裡？

- 功能完整嗎？引導做好了嗎？付費流程順暢嗎？

你要**模擬**，不是列清單。真正去感受哪裡有摩擦。
